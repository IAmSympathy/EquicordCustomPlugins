# Plugins Personnalisés Equicord

Ce repository contient mes plugins personnalisés pour Equicord.

## Installation

Ces plugins sont automatiquement installés via le script Install or Update Equicord.ps1.

## Développement

Pour ajouter un nouveau plugin :
1. Créez un nouveau dossier avec le nom du plugin
2. Ajoutez un fichier index.tsx avec le code du plugin
3. Committez et pushez les changements
